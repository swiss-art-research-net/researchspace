i18n Component

Basic implementation for multilingual applications.

The component reads a JSON dictionary for messages in different languages. A default message file is required. Additional languages can be added.

Configuration

Prepare a `messages.json` file with the following structure that contains the messages in the default language. The file shold be stored in /assets/i18n/messages.json. Prepare an additional `messages_{language}.json` file for each additional language, where {language} is the language code. The file shold be stored in /assets/i18n/messages_{language}.json. e.g. `messages_de.json` for German.

Usage

Call the component with the message key as the content of the component. The component will display the message in the set language.

HTML5 Semantic Components

All semantic components are custom HTML5 Web Components, which operate on the result of SPARQL queries (usually either SPARQL SELECT or CONSTRUCT) being executed over the knowledge graph.

Components differ in their configuration as well as as hooks for customization. In particular, they provide different levels of abstraction (and flexibility respectively) in how the tuple- or graph results are being processed.

HTML5 Markup Components

Markup components are custom HTML5 Web Components that allow for formatting and structuring the content of pages and templates, providing (interactive) functionality that goes beyond what native HTML markup elements support (e.g. interactive toggling, tabs, grid elements and so forth).

Some of these components are provided through the integration of existing libraries such as react-bootstrap, whereas others being custom developments.

React-Bootstrap

Components provided through react-bootstrap are prefixed with bs-*.
  • Vertical Tabs
  • Horizontal Tabs
  • Grid Layout
  • Alert
  • Navbar and Navbar Items
  • Panels
  • Popover

ResearchSpace platform

Components provided through the ResearchSpace platform are prefixed with mp-*.
  • Modal Dialog
  • Resource Link
  • Resource Thumbnail
  • (Lazy) Page Loader
  • Text Truncate
  • Sparql Result Download
  • Split Pane
  • CSS Layout
